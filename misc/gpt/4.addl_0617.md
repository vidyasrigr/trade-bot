One thing I would add to his list

A Stage 3.5.

Confidence Calibration

For every recommendation:

Store:

conviction
expected success probability
actual result

Then monthly:

70 conviction
→ won 68%

80 conviction
→ won 79%

90 conviction
→ won 61%

If 90-conviction trades only win 61%:

the model is overconfident.

That information becomes incredibly valuable later.
